📖 Sobre o Projeto
Este projeto apresenta a utilização do Bootstrap, um dos frameworks de front-end mais populares para desenvolvimento web. O Bootstrap, ele fornece um conjunto de componentes prontos, sistema de grid responsivo e classes utilitárias, permitindo criar interfaces modernas, organizadas e adaptáveis a diferentes dispositivos.
Ao usar o Bootstrap, é possível desenvolver páginas web de forma mais rápida e eficiente, utilizando layouts responsivos e componentes visuais que seguem boas práticas de design.

⚙️ Funcionalidades e Conceitos Aplicados

Durante o desenvolvimento, foram aplicados os seguintes conceitos:

    Bootstrap 5
    Sistema de Grid Layout responsivo
    Uso de Containers, Rows e Columns
    Componentes prontos como Navbar, Buttons e Cards
    HTML Semântico
    Estrutura organizada com tags como <header>, <main> e <section>
    Responsividade
    Adaptação automática para desktop, tablet e mobile
    rsionamento
    Controle de versão utilizando Git e GitHub.
